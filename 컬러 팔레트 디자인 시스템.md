## 컬러 사용 가이드

### 1. Primary Color — 비비드 블루 (`#4761E8`)

| 사용 위치 | 예시 |
|-----------|------|
| CTA 버튼 | 회원가입, 문의하기, 다운로드 |
| 링크 텍스트 | 인라인 링크, 네비게이션 활성 상태 |
| 강조 요소 | 선택된 탭, 진행 바, 포커스 링 |
| 아이콘 | 주요 기능 아이콘 |

```css
.btn-primary {
  background-color: var(--color-button-primary);
  color: var(--color-text-inverse);
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  transition: background-color 0.2s ease;
}

.btn-primary:hover {
  background-color: var(--color-button-hover);
}

.tag--primary {
  background-color: var(--color-primary);
  color: #FFFFFF;
  padding: 4px 12px;
  border-radius: 16px;
  font-weight: 600;
}
```

### 2. Secondary Color — 다크 차콜 (`#2A2A27`)

| 사용 위치 | 예시 |
|-----------|------|
| 메인 타이틀 | h1, h2 제목 텍스트 |
| 네비게이션 바 | 고정 상단 바 배경 |
| 푸터 | 전체 푸터 영역 배경 |
| 본문 텍스트 | 일반 paragraph 텍스트 |

```css
.section-title {
  color: var(--color-secondary);
  font-weight: 700;
}

.navbar {
  background-color: var(--color-bg-dark);
  color: var(--color-text-inverse);
}

.footer {
  background-color: var(--color-bg-dark);
  color: var(--color-text-inverse);
}
```

### 3. Accent Color — 웜 앰버 (`#C47B2B`)

| 사용 위치 | 예시 |
|-----------|------|
| 배지/라벨 | "NEW", "HOT", "Featured" 태그 |
| 알림/경고 | 토스트 메시지, 경고 배너 |
| 보조 CTA | 덜 중요한 액션 버튼 |
| 장식 요소 | 밑줄 강조, 아이콘 포인트 |

```css
.badge--accent {
  background-color: var(--color-accent);
  color: #FFFFFF;
  padding: 2px 10px;
  border-radius: 12px;
  font-size: 0.75rem;
  font-weight: 600;
}

.highlight-underline {
  border-bottom: 3px solid var(--color-accent);
  padding-bottom: 2px;
}
```

### 4. Interactive Colors — 버튼 & 링크

```css
/* 아웃라인 버튼 */
.btn-outline {
  background-color: transparent;
  color: var(--color-primary);
  border: 2px solid var(--color-primary);
  padding: 12px 24px;
  border-radius: 8px;
  transition: all 0.2s ease;
}

.btn-outline:hover {
  background-color: var(--color-primary);
  color: #FFFFFF;
}

/* 앰버 버튼 (보조 CTA) */
.btn-accent {
  background-color: var(--color-accent);
  color: #FFFFFF;
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  transition: background-color 0.2s ease;
}

.btn-accent:hover {
  background-color: var(--color-accent-dark);
}

/* 링크 */
a {
  color: var(--color-link);
  text-decoration: none;
  transition: color 0.2s ease;
}

a:hover {
  color: var(--color-link-hover);
}
```

### 5. 배경 색상 사용 패턴

```css

/* 카드/패널 */

  background-color: var(--color-bg-white);

  border-radius: 12px;
}

/* 히어로/푸터 */
.hero, .footer { background-color: var(--color-bg-dark); }
```

## 반응형 고려사항

### 다크모드

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* Background — 다크 차콜 기반 */
    --color-bg-primary: #1E1E1C;
    --color-bg-secondary: #2A2A27;
    --color-bg-white: #333330;
    --color-bg-dark: #151513;

    /* Text — 밝은 톤으로 전환 */
    --color-text-primary: #E9E6E3;
    --color-text-secondary: #B5B2AF;
    --color-text-muted: #7A7A76;
    --color-text-inverse: #1E1E1C;

    /* Primary — 밝기를 높여 어두운 배경에서 가독성 확보 */
    --color-primary: #6B80F0;
    --color-primary-light: #8B9CF5;
    --color-primary-dark: #4761E8;

    /* Accent — 밝기 조정 */
    --color-accent: #D99A4E;
    --color-accent-light: #E5B06E;
    --color-accent-dark: #C47B2B;

    /* Interactive */
    --color-button-primary: #6B80F0;
    --color-button-hover: #8B9CF5;
    --color-link: #6B80F0;
    --color-link-hover: #8B9CF5;

    /* Border */
    --color-border-light: #3D3D39;
    --color-border-accent: #6B80F0;

    /* Shadow — 더 진하게 */
    --shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.3);
    --shadow-md: 0 4px 12px rgba(0, 0, 0, 0.4);
    --shadow-lg: 0 8px 24px rgba(0, 0, 0, 0.5);
  }
}
```

### 화면 크기별 고려사항

| 화면 크기 | 권장 사항 |
|-----------|-----------|
| **모바일 (< 768px)** | 카드 그림자 축소(`--shadow-sm`), 버튼 풀 와이드 적용, 앰버 포인트 요소 크기 축소 |
| **태블릿 (768-1024px)** | 카드 2열 그리드, 네비게이션 간소화 |
| **데스크탑 (> 1024px)** | 풀 디자인 적용, 그라디언트 및 그림자 효과 활용 |

### WCAG 접근성 대비율 체크

| 조합 | 대비율 | 판정 |
|------|--------|------|
| `#2A2A27` on `#E9E6E3` | 10.8:1 | AAA 통과 |
| `#2A2A27` on `#FFFFFF` | 12.8:1 | AAA 통과 |
| `#FFFFFF` on `#4761E8` | 4.6:1 | AA 통과 |
| `#FFFFFF` on `#2A2A27` | 12.8:1 | AAA 통과 |
| `#FFFFFF` on `#C47B2B` | 3.2:1 | AA Large 통과 |
| `#4761E8` on `#E9E6E3` | 4.2:1 | AA Large 통과 |

> 앰버(`#C47B2B`) 위 흰색 텍스트는 대비율 3.2:1로 **큰 텍스트(18px+ 또는 14px bold)에만 사용** 가능. 작은 텍스트에는 다크 차콜(`#2A2A27`) 배경 + 앰버 텍스트 조합을 권장합니다.
